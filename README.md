# Enterprise Network Security Assessment (WBMUD Simulation)

## 📌 Overview
This repository contains a simulated enterprise network security assessment built using Cisco Packet Tracer. The network includes segmented IT, OT, and Remote environments with routing, connectivity testing, and protocol security analysis.

## 🧠 Skills Demonstrated
1. Network design and segmentation (IT, OT, Remote environments)
2. IP addressing and static routing configuration
3. Protocol security analysis (TCP/UDP, HTTP/HTTPS, SSH)


## 🗂️ Repository Structure
`README.md` — Project overview and documentation

`screenshots/` — Network topology, routing, connectivity tests, and protocol demonstrations.


## 🔎 Key Takeaway
This project demonstrates how proper network segmentation and protocol selection improve security and control in enterprise environments.
